Dhruvesh Nihalani Co-op Report Summer 2025

Introduction

Hi, my name is Dhruvesh Nihalani, and I am a fourth-year Computer Science student at the University of Guelph. This document highlights my experience as a Full Stack Developer Co-op at Ellisdon. The website is divided into different segments: job description, goals, and conclusion.

Job Description

As a Co-op Full Stack Developer, you will:

  • Design, build, test, and document modern front-end UI/UX applications using React/Redux and functional microservices with NodeJS or Spring Boot.
  • Deploy and monitor microservices using cloud infrastructure.
  • Collaborate with architects, UX designers, product owners, QA, and DevOps to prioritize delivering value to customers in an Agile environment.
  • Contribute to improving product quality and the development process.

This is the right role if you have:

  • Strong understanding of object-oriented programming (OOP) concepts.
  • Experience with frontend frameworks like React/Redux.
  • Experience with backend frameworks such as NodeJS or Java Spring Boot.
  • Understanding of RESTful API implementation.
  • Experience with relational databases like MySQL or NoSQL like MongoDB.
  • Excellent interpersonal and communication skills.
  • Experience with Docker and Kubernetes on public cloud platforms (GCP, Azure, AWS).

Goals

Goal 1: Learn more about agile principles

I attended daily standups, backlog refinement, sprint planning, retrospectives, and reviews. Through this, I learned about story pointing, being flexible, valuing teamwork, and ensuring that software meets customer needs.

Goal 2: Learn more about React and frontend development

I learned about React, including hooks and state management. I implemented a custom rich text editor with Quill.js, including inline image uploads in our application.

Goal 3: Learn more about Node.js and backend development

I implemented five API endpoints, including a PATCH endpoint for soft-deleting files from a GCP bucket, a reverse proxy service for file downloads, and Redis caching to improve performance.

Conclusion

In summary, by participating in team meetings and activities, I learned a lot about Agile principles. I built features using React and improved my backend development skills through practical experience.
